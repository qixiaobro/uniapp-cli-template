## 安装依赖
```
npm install
```
在
### 运行项目
```
npm run dev:%PLATFORM%
```
### 发布项目
```
npm run build:%PLATFORM%
```

|  值   |平台  |
|  ----  | ----  |
| h5  | H5 |
| mp-alipay  | 支付宝小程序 |
